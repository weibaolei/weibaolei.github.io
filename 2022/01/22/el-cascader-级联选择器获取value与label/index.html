el-cascader 级联选择器获取value与label

发表于 2022-01-22
  • 深海の.大菠萝
首页 工作

在使用 element-ui 的 el-cascader 级联选择器时，我想在 @change 函数中获取当前选中的 value 与 label 值，但是使用的原有参数只能获取 value 值。

可以使用 that.$refs[“cityCascader”].getCheckedNodes()[0].pathLabels 函数来获取当前的 label 值。
